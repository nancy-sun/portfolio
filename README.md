## Description
This is my personal portfolio project, experimenting Three.js. 


## Install  
#### clone
``` $ git clone https://github.com/nancy-sun/portfolio ```
#### install
```$ npm install```

#### run
```$ npm start```

## Tech Stack
- React
- CSS(Sass)
- Three.js (react-three-fiber)
- Redux
- MUI
